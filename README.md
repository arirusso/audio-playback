# Audio Playback

## Installation

These packages must be installed first:

* portaudio
* libsndfile

Install the gem using

    gem install audio-playback

Or if you're using Bundler, add this to your Gemfile

    gem "audio-playback"

## Usage

From the command line

    playback something.wav
